# PeopleHub
PeopleHub - Microfrontend HR Platform
- Designed and Built a microfrontend architecture using single-spa with angular 20
- Implemented cross app communication using custom browser events and shared JWT authentication via single spa custom props.
- Built RESTful backend with Nodejs,Express and MongoDB supporting role-based access control.
- Achieved zero reload navigation between microfrontends using SystemJS module loading and layout engine routing.


Tech: Angular 20, single-spa, Typescript, NodeJS, Express, MongoDB, JWT, SystemJS.
